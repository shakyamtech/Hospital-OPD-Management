# OPD Connect - Hospital Management System

OPD Connect is a web application designed to manage hospital Outpatient Department (OPD) bookings, doctors' schedules, and patient visits.

## Tech Stack

### Frontend
- **HTML5**: Structured semantic web page markup.
- **CSS3 (Vanilla)**: Layouts, custom styling variables, and responsive design.
- **JavaScript (Vanilla)**: Frontend core application logic, client-side routing, and API communication.
- **Google Fonts (Inter & Material Symbols)**: High-quality typography and healthcare-focused icons.

### Backend
- **Python**: Programming language used for server-side endpoints.
- **FastAPI**: Modern, high-performance web framework for API routing.
- **Uvicorn**: ASGI web server to run the FastAPI application.
- **Pydantic**: Data modeling, request body parsing, and type-validation.
- **Firebase Admin SDK**: Server-side administration for Firebase Services.

### Database & Auth
- **Google Cloud Firestore**: NoSQL cloud database for storing doctor listings, department slots, and booking schedules.
- **Firebase Authentication**: User authentication and doctor/admin credential management.
